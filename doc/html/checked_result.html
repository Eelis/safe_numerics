Description

checked_result is a wrapper class designed to hold the result of some operation. It can hold either the result of the operation or information on why the operation failed to produce a valid result. Note that this type is an internal feature of the library and is not explicitly exposed to users. But it is important to know about this of one wants to extend the application of the library to other types. It may be useful in other applications.

Template Parameters

The sole template parameter is the return type of some operation.

Parameter Description
R Type of value expected if there is not error to be reported

Notation

Symbol Description
R Type of value expected if there is not error to be reported
r An instance of type R
c an instance of checked_result<R>
t an instance of checked_result<T> for some type T not necessarily the same as R
e An instance of type safe_numerics_error

Valid Expressions

All expressions are constexpr.

Expression Return Type Semantics
checked_result(r) checked_result<R> constructor with valid instance of R
checked_result<R>(t) checked_result<R> constructor with checked_result<T> where T is not R. T must be convertible to R.
checked_result(e, msg) checked_result<R> constructor with error information
static_cast<R>(c) R extract wrapped value - asserts if not possible
static_cast<e>(c) safe_numerics_error extract wrapped value - asserts if not possible
static_cast<const char *>(c) const char * returns pointer to error message
c < t
c >= t
c > t
c <= t
c == t
c != t
 boost::logic::tribool compare the wrapped values of two checked_result instances. If either one contains an error_type value, return boost::logic::tribool::indeterminant.
c.exception() bool true if checked_result contains an error condition

Example of use

#include <boost/numeric/safe_numerics/checked_result.hpp>

template<class R>
checked_result<R>
constexpr modulus(
    const R & t,
    const R & u
) {
    if(0 == u)
        return checked_result<R>(
            exception_type::domain_error,
            "denominator is zero"
        );

    return t % u;
}
